Bring Your Own Model

CodeCanvas v2 is open-source and architected to work entirely offline. Here is how to link your local AI environment.

1

Install Ollama

Ollama allows you to run large language models locally on your Mac, Windows, or Linux machine without relying on corporate clouds or APIs.

Visit Ollama.com, download the installer for your OS, and run the setup.

2

Pull Your Preferred Model

Open your terminal and pull the foundational model you want the CodeCanvas AI Pipeline to use. We highly recommend models optimized for code generation.

$ ollama run gemma:7b

Gemma (Google)

Fast, accurate, and lightweight. Excellent for prompt interpretation.

ollama run gemma

CodeLlama (Meta)

The industry standard for generating strict HTML/CSS formats.

ollama run codellama

DeepSeek Coder

Advanced logic capabilities for complex creative structures.

ollama run deepseek-coder

Llama 3

Incredibly capable conversational model for rich text generation.

ollama run llama3
3

Configure CodeCanvas

Once your chosen model is running in your terminal, open your CodeCanvas project directory and navigate to api/generate.php.

Locate the ollamaRequest function around line 90 and update the model string to map exactly to the model you pulled.

curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode([
    'model' => 'gemma:7b', // Change this to match your local model
    'prompt' => $prompt_text,
    'stream' => false,
]));

You are now running a 100% private, locally deployed AI portfolio generator.

Guidelines and rules for using the CodeCanvas platform.

Ownership of Generated Code

CodeCanvas v2 provides intelligence layer mapping. Any code (HTML/CSS/JS) generated by the AI engine utilizing your prompts belongs entirely to you. You are free to export, modify, or host the code commercially without attribution.

Acceptable Use

You agree not to use CodeCanvas generative components to construct highly malicious templates (e.g. phishing layouts, credential harvesting interfaces), knowing that underlying AI guardrails may intervene and terminate sessions.

Rate Limits

To preserve backend stability via proxy APIs, users are restricted to standard concurrent generation quotas. Repeated abuse or artificial generation loops may trigger a temporary cooldown.
